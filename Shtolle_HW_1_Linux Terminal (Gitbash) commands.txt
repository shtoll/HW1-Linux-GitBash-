1) Посмотреть где я 
pwd

2) Создать папку
mkdir название папки

3) Зайти в папку 
cd название папки

4) Создать 3 папки 
mkdir название папки название папки название папки

5) Зайти в любоую папку 
cd название папки

6) Создать 5 файлов (3 txt, 2 json) 
touch t1.txt t2.txt t3.txt j4.json j5.json

7) Создать 3 папки 
mkdir ff1 ff2 ff3

8. Вывести список содержимого папки
ls -la

9) + Открыть любой txt файл
vim t1.txt

10) + написать туда что-нибудь, любой текст.
жмем i, затем enter, затем вводим текст любой

11) + сохранить и выйти. жмем ESC вводим:
:wq

12) Выйти из папки на уровень выше
cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку.
cd f1
mv j4.json j5.json ff2

14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
cp t1.txt t2.txt ff2

15) Найти файл по имени
find t1.txt

16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
tail -f t1.txt 
tail -f t1.txt | grep world (поиск слова world в документе)

17) вывести несколько первых строк из текстового файла
head -5 t1.txt ( по умолчанию идёт первых 10 строк, я ввёл значение 5)

18) вывести несколько последних строк из текстового файла
tail n t1.txt ( по умолчанию выводит 10 последних строк, я ввёл значение 5)
tail -n5 t1.txt

19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
less t1.txt (f вперед на стр, b назад на стр, -N покажет номера строк)

20) вывести дату и время
date
